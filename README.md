# Caesar Cipher Encryption & Decryption Tool

## 📌 Overview

This project is a Python-based Caesar Cipher application developed as part of the SkillCraft Technology Cyber Security Internship (Task 01).

The program allows users to encrypt and decrypt messages using the Caesar Cipher algorithm by shifting alphabetic characters by a specified number of positions.

---

## 🚀 Features

* Encrypt text using Caesar Cipher
* Decrypt encrypted text
* Supports uppercase and lowercase letters
* Preserves spaces, numbers, and special characters
* Validates user input for shift values
* Simple command-line interface

## ▶️ How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

```bash
python main.py
```

4. Enter the message.
5. Enter the shift value.
6. Choose:

   * 1 → Encrypt
   * 2 → Decrypt
